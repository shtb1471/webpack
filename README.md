# 基本安装
  首先我们创建一个目录，初始化 npm，以及在本地安装 webpack：
  ```
  mkdir webpack-demo 
  cd webpack-demo
  npm init -y
  npm install --save-dev webpack
  ```
# 加载css
  ```
  npm install --save-dev style-loader css-loader
  ```
# 加载图片
  ```
  npm install --save-dev file-loader
  ```
加载数据
  ```
  npm install --save-dev csv-loader xml-loader
